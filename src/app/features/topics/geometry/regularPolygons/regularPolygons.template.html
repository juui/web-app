Diagonal de un polígono

El segmento que une dos vértices no consecutivos del polígono se denomina diagonal. En este pentágono, se pueden trazar \(5\) diagonales: \( \overline { AC } \), \( \overline { AD } \), \( \overline { BD } \), \( \overline { BE } \), \( \overline { CE } \).

Número de diagonales

I) A partir de un vértice de un polígono de \(n\) lados se pueden trazar \(n-3\) diagonales.

II) La fórmula para calcular el total de diagonales de un polígono de \(n\) lados es: $$ \frac { n(n-3) }{ 2 } $$

Ángulos internos

Son los ángulos formados por \(2\) lados consecutivos y su interior.

Ángulos externos

Son los ángulos adyacentes a los ángulos internos que se obtienen al prolongar los lados en un mismo sentido. Dada la siguiente figura:

I) Ángulos externos: \( \alpha \), \( \theta \) y \( \omega \).

II) Ángulos externos: \( { e }_{ 1 } \), \( { e }_{ 2 } \) y \( { e }_{ 3 } \).

III) Las parejas de ángulos consecutivos (interno y externo) son complementarios, es decir: \( { e }_{ 1 } + \alpha = 180° \), ... , \( { e }_{ 3 } + \omega = 180° \).

Centro de un polígono regular

El centro de un polígono regular es el centro común de sus circunferencias inscrita y circunscrita. En la figura se representa con la letra \(O\).

Radio de un polígono regular

El segmento \(r\) que une el centro \(O\) con un vértice del polígono se denomina radio.

Apotema de un polígono regular

El segmento de longitud \(a\) que une el centro \(O\) con el punto medio de uno de los lados del polígono.
